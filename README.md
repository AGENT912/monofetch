# What's monofetch and why did I made it?
I always didn't like fastfetch. It looked too heavy and too popular. And it didn't support a lot of linux distributions. As an example, on RaspberrypiOS it showed Debian logo instead of RaspberryPiOS logo. And this happened to most linux distros, that aren't in the "MOST POPULAR DISTROS" list. So, I made this. This fetch doesn't use any imports(even import os isn't allowed here). The only import is colorama, but I should soon replace it with custom colour codes.

# How to use it?
First, download "monofetch.py". Then, type "python3 qqqq/monofetch.py", where "qqqq" is your downloads directory. Soon, I should make an installer for this, so you can run it by only type "monofetch" in your terminal.
